# 達爾文核心標準

{:.lead}
達爾文核心標準是由[達爾文核心標準維護小組](https://www.tdwg.org/standards/dwc/#maintenance-group)所維護的標準。 其中包含一份術語表（在其他情境中這些術語可能被稱為屬性、元素、欄位、欄位、屬性或概念），旨在透過提供識別碼、標籤與定義，**促進生物多樣性相關資訊的共享**。 達爾文核心標準主要基於分類群，以及透過觀察、標本、樣本及相關資訊所記錄在自然界的分布狀況。
