# Darwin Core

{:.lead}
Darwin Core es un estándar para compartir datos sobre biodiversidad, mantenido por el [Grupo de Mantenimiento de Darwin Core](https://www.tdwg.org/standards/dwc/#maintenance-group). Incluye un modelo conceptual, especificaciones y esquemas para compartir datos mediante diversos formatos, así como un glosario de términos (que en otros contextos podrían denominarse propiedades, elementos, campos, columnas, atributos o conceptos) mediante la provisión de identificadores, etiquetas y definiciones.
